# Polic-

A project focused on police exam preparation and study support.

## Features
- PDF-based revision support
- Mock tests and practice sets
- Important topics categorization
- Study material organization
- Simple and user-friendly interface

## Goal
Help students prepare efficiently with revision tools, practice content, and structured resources.

## Usage
1. Clone the repository
2. Add study materials
3. Run the project setup
4. Start learning and practicing

## Future Improvements
- AI-based revision
- Performance analytics
- Personalized recommendations
- More practice modules
